

# Library Inventory Manager

This is a simple command-line Python program to manage a library's book inventory. It helps to add books, issue and return them, and view all books. The program saves book data to a file so the information is kept between uses.

## Features
- Add new books with title, author, and ISBN
- Issue books to users (change status to issued)
- Return issued books (change status to available)
- View all books in the library
- Data saved and loaded from a file

## How to Use
1. Run the program.
2. Choose an option from the menu:
   - Add Book
   - Issue Book
   - Return Book
   - Show All Books
   - Exit
3. Follow the prompts to enter book details or ISBN as needed.

## Purpose
This project shows object-oriented design using classes, file handling, and a simple text menu interface.

## Files
- `lab3.py`: Main code file with Book class and library menu.
- `books.txt`: File where book data is saved.

## Requirements
- Python 3
